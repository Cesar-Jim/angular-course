JSON

Aplicando el pipe json de la forma: "personaje | json", convierte el objeto a un JSON válido SIN formato:

{{ personaje | json }}


Aplicando el pipe json de la forma: "personaje | json" con uso del tag pre (bootstrap), el objeto se convierte a un JSON válido CON formato:

        
{{ personaje | json }}
